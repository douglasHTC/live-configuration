腾讯云后台配置

腾讯云后台配置涉及直播推流地址与聊天室的配置。腾讯云地址: https://www.tencentcloud.com/zh。账号需要找运维roy开通子账号。

直播配置

  1. 腾讯云搜索云直播

  2. 点击马上开始到云直播管理

  3. 在常用工具-地址生成器中配置推流地址与播放地址。地址类型需要选择推流和播放地址组

  4. 配置推流地址信息

    配置 说明
    推流域名 选择推流域名，测试需要选择测试环境使用的域名
    播放域名 选择播放域名，测试需要选择测试环境使用的域名
    AppName 每个直播间该值保持唯一
    StreamName 每个直播间该值保持唯一
    加密类型 可使用默认
    过期时间 设置生成推流地址的过期时间，到期后无法再使用改地址
    转码模板 保持默认配置

  5. 生成地址
    配置好后需要使用到的地址为：
    1. 推流地址中的OBS服务器、OBS推流码：分别配置在管理台直播间的配置项；WebRTC地址：用于OBS直播软件的直播令牌

    2. 播放地址中的4个地址，分别配置在管理台直播间的播放地址配置项。C端实际播放地址只使用到最后一个webrtc地址

    在近期记录中可以查看当前账号近期生成的地址信息（本地缓存）

聊天室配置

  1. 腾讯云搜索即时通信IM

  2. 点击立即开始体验进入到IM管理

  3. 选择Applications-hotcoin，目前测试的直播间都是加在hotcoin应用下，后续可能调整

  4. 选择chat进入

  5. 选择Groups进入，在这里添加聊天室

  6. 配置聊天室

    配置 说明
    Group Name 任意填入，自己可区分即可
    Group Owner ID 可不填
    Group Type 选择Audio-Video Group

  7. 生成聊天室后在列表中找到刚生成的聊天室ID，一般在最后一页。需要使用的Group ID，将其配置在管理台直播间中的聊天室ID。

    聊天室与直播间必须是一对一关系，一个聊天室必须只能配给一个直播间

OBS直播配置

OBS直播软件，用于推流直播内容，下载地址https://obsproject.com/download

  1. 进入OBS直播配置
  2. 配置推流信息
    1. 服务选择WHIP
    2. 服务器地址使用默认地址https://webrtcpush.tlivewebrtcpush.com/webrtc/v2/whip。默认地址无法直播时可切换使用备用地址https://webrtcpush.tlivewebrtcpush2.com/webrtc/v2/whip
    3. Bearer令牌填入云直播生成地址中的WebRTC推流地址
  3. 配置直播源
    在来源中可以添加多种类型的直播源

    测试时可使用浏览器直播源，任意配置一个带有视频的url即可
  4. 开始推流
    点击开始直播即可开始推流

管理台配置

  1. 直播管理入口在活动管理-直播管理
  2. 配置直播间
    用户ID，直播间名称等通用配置项不再赘述，只介绍推流相关配置
    1. 推流地址中OBS服务器与推流码分别配置为云直播后台中生成的OBS地址
    2. 播放地址分别配置为云直播后台中生成的播放地址。前三个可以任意配置，webrtc必须使用推流地址下的webrtc播放地址。
    3. 聊天室ID配置为IM中生成的Group ID
